# Rise 360 Lesson Blocks Reference

This file summarizes the content blocks available in **Articulate Rise 360** that are used to build responsive lessons.

---

## 🧱 1. Text & Content Blocks (Foundation)
Used to deliver core instructional content.

**Common blocks**
- Paragraph
- Heading
- Statement (note, tip, warning)
- Quote
- Bullet & numbered lists
- Table
- Columns

**Best for:** explanations, instructions, key points, summaries.

---

## 🖼️ 2. Media Blocks
Used to add visuals and multimedia to support learning.

**Common blocks**
- Image
- Image gallery
- Video
- Audio
- Embedded web content

**Best for:** visual context, demonstrations, accessibility, diverse learning styles.

---

## 🔄 3. Interactive Blocks (Engagement & Exploration)
Designed to promote learner interaction and active exploration.

**Common blocks**
- Accordion
- Tabs
- Flashcards (stack & grid)
- Labeled graphic (hotspots)
- Process (step-by-step)
- Timeline
- Sorting activity
- Scenario interactions
- Buttons

**Best for:** chunking content, comparisons, guided exploration, step-by-step workflows.

---

## ✅ 4. Knowledge Check & Assessment Blocks
Used for formative assessment within lessons.

**Common blocks**
- Multiple choice
- Multiple response
- Fill-in-the-blank
- Matching
- Sorting activities

**Best for:** reinforcing learning, providing feedback, checking understanding.

---

## 🧭 5. Divider & Navigation Blocks
Used to structure lessons and guide learner progression.

**Common blocks**
- Divider
- Continue block
- Button (link to next lesson or external resource)

**Best for:** pacing, section separation, navigation control.

---

## 🧩 How Blocks Work Together
Think of blocks as modular components that can be stacked in any order.

**Example combination:**
- Heading + paragraph → introduce a concept
- Image → provide visual context
- Tabs → compare ideas
- Process → show workflow
- Knowledge check → reinforce learning
- Continue block → move to next lesson

---

## 🎯 Example Lesson Structure
1. Heading + paragraph (introduction)
2. Image block (visual context)
3. Tabs block (compare concepts)
4. Process block (step-by-step workflow)
5. Knowledge check (quiz)
6. Continue block (navigation)

---

*Use this reference when designing Rise 360 lessons to ensure a balanced mix of content, media, interaction, and assessment.*
